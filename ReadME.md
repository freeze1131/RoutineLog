# RoutineLog

RoutineLog is a simple SwiftUI app for tracking daily routines, notes, and personal habits.

The goal of this project is to build a small but polished iOS app that can be published on the App Store and also used as a portfolio project on GitHub.

## Status

In development.

## Features

Planned features for the first version:

- Daily routine logging
- History screen
- Weekly statistics
- Local data storage
- Daily reminders
- Clean and simple SwiftUI interface

## Tech Stack

- Swift
- SwiftUI
- SwiftData
- Charts
- Local Notifications

## Project Goal

This app is being developed as a real App Store project, not just a demo.  
The main focus is to keep the app simple, useful, and easy to maintain.
